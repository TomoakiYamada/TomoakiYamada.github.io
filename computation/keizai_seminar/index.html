定量的マクロ経済学と数値計算@経済セミナー

Sagiri Kitao, Takeki Sunakawa, Tomoaki Yamada
Jan 27, 2020
Code

『経済セミナー』で2018年12月・2019年1月号から2020年2・3月号まで合計8回にわたって、「定量的マクロ経済学と数値計算」という連載講義を行いました（北尾早霧氏・砂川武貴氏と共著）。 連載で使用したすべてのコードはGitHubからダウンロードすることができます。 主にMatlabで書かれていますが、一部、PythonやJuliaも使用しています。
