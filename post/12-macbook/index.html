備忘録：MacBook Airに引っ越し

Tomoaki Yamada
Oct 20, 2022

MacBook Airを新規購入

  • MacBook Air（M2チップ）を新規購入して移行ツールを使わないで新規セットアップをしたので、インストールしたもの一式を備忘録を兼ねてまとめる

最初にやる事

  • Google関連➔2段階認証が必要なのでスマホを片手にセットアップ
    • メール設定時に自動で要求される
    • Appleアカウントにログインしているので勝手に一部設定されている?
  • Google日本語入力をインストール➔M2チップなのでRosettaをインストールするように促される
    • Google認証
  • Chromeをダウンロードしてインストール
    • Google認証が必要(3度目)

ダウンロード：基本的にフリーのもの

  • Anaconda
  • DeepL
  • Fork
  • GitHub Desktop
  • GitKraken
    • GitHubをprivateで使う場合は有料
  • Grammarly
    • 左下にあるAppsからDesktop版とブラウザ版をインストール
  • Homebrew
    • Hugo
  • iTerm2
  • Julia
  • JupyterLab Desktop
  • Karabiner-Elements
    • 円マークとバックスラッシュを入れ替えるために使う
  • Logi Options
    • 外部モニタに接続してマウスを使う場合に必要
  • LyX
    • サーバーのせいかダウンロードに時間がかかる
  • MacTeX
  • Notion
  • PyCharm
    • JetBrainはFleetという新しいIDEを開発している
  • Skim
  • Skype
  • R
  • RStudio
  • VS Code
    • インストール後の初回起動時に日本語パックをインストールするように促される
  • Zoom
    • なぜかZoomのインストーラは見つけにくいので直リン
  • Zotero
    • 文献管理

Fontを追加する

  • SourceCodePro
  • SourceHanCodeJP
  • HackGenNerd
  • IPAexfont
  • RictyDiminished

有料（大学ライセンスを含む）

  • Microsoft Office
  • Matlab
  • Stata
  • Alfred5

App Storeからダウンロード：有料のものを含む

  • Commander One PRO
  • Cyberduck
  • GoodNotes
  • LINE
  • Magnet
  • NoteShelf
  • PopClip
  • Slack
  • Tot

その他

  • Beamerのオリジナルテンプレートを配置
